How to Use These Prototypes

  1. Serve locally: python -m http.server 3000
  2. Open in browser: http://localhost:3000
  3. Convert to Figma: Use html.to.design Figma plugin
  4. Test RTL: Click the عربي button to switch to Arabic/RTL layout

Design System Files

  • design-tokens.json - Colors, typography, spacing, shadows
  • css/styles.css - Complete design system CSS with RTL support
  • js/app.js - Language switching, dropdowns, modals, toasts
