# Liyaqa

Gym & Fitness Club Management Platform - A complete SaaS solution for managing memberships, attendance, billing, and scheduling.

## Features

- Multi-tenant architecture (Organization → Club → Location)
- Member management with subscriptions
- Attendance tracking (QR code, manual, card)
- Class scheduling and bookings
- Invoice generation with PDF export
- Bilingual support (English/Arabic)
- Platform admin dashboard (B2B)

## Tech Stack

**Backend:** Spring Boot 4.0 / Kotlin / PostgreSQL
**Frontend:** Next.js 15 / React / TypeScript / Tailwind CSS
**Mobile:** Kotlin Multiplatform (Android/iOS)
