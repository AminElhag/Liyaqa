## Technical Specs

### File Formats
- **Vector:** SVG (infinitely scalable, web-optimized)
- **Format:** XML-based, browser-native
- **Size:** 1-3KB per file (lightweight)
- **Compatibility:** All modern browsers, design tools

### Grid System
- **Base:** 8px grid
- **Safe zone:** 20% padding
- **Alignment:** Pixel-perfect at all sizes

### Minimum Sizes
| Logo Type | Minimum Width | Minimum Height |
|-----------|---------------|----------------|
| Horizontal | 120px | ~34px |
| Vertical | 100px | 100px |
| Icon-only | 32px | 32px |
| Favicon | 16px | 16px |

### Stroke Specifications
- **Main calligraphy:** 2-2.5px
- **Fine details:** 1.5px
- **Caps:** Round
- **Joins:** Round
