# Liyaqa Admin Guide

**Guide for Gym Administrators & Staff**

---

##  Welcome, Administrator!

This guide covers daily operations and administrative tasks for managing your gym with Liyaqa.

---

## Quick Start

**Access Admin Portal**: `{your-gym}.liyaqa.com`
**Login**: Use your admin credentials
**Role**: Club Admin, Manager, or Staff

---

## Dashboard Overview

Your dashboard shows:
- **Hero Stats**: Active members, monthly revenue, today's attendance, occupancy
- **Revenue Trend**: Last 30 days with comparison
- **Activity Timeline**: Recent actions (signups, bookings, payments)
- **Quick Actions**: Add member, create invoice, manual check-in

---

## Member Management

### Add New Member
1. Dashboard > **Members** > **Add Member**
2. Enter details: Name, email, phone, date of birth
3. Select membership plan
4. Set start date
5. Click **Create Member**
6. Member receives welcome email with login credentials

### Search & Filter Members
- **Search**: By name, email, phone, or member ID
- **Filter**: By status (Active, Frozen, Expired), plan, join date
- **Sort**: By name, join date, expiry date

### View Member Profile
Click on any member to see:
- Personal information
- Current subscription and history
- Attendance records
- Bookings
- Payments and invoices
- Notes and activity log

### Freeze Member
1. Open member profile
2. Go to **Subscriptions**
3. Click **Freeze**
4. Select freeze period
5. Confirm

### Cancel Membership
1. Open member profile
2. Go to **Subscriptions** > **Cancel**
3. Select cancellation reason
4. Set effective date
5. Calculate any refunds or fees
6. Confirm cancellation

---

## Subscription Management

### Create Subscription
1. Open member profile or go to **Subscriptions** > **New**
2. Select member
3. Choose membership plan
4. Set start date
5. Enable/disable auto-renewal
6. Click **Create**

### Renew Subscription
1. Go to **Subscriptions**
2. Filter by "Expiring Soon"
3. Select subscription
4. Click **Renew**
5. Confirm new period

### Upgrade/Downgrade
1. Open subscription
2. Click **Change Plan**
3. Select new plan
4. Review proration
5. Confirm change

---

## Class & Schedule Management

### Create Class Type
1. **Classes** > **Class Types** > **Add New**
2. Enter: Name, description, duration, capacity, difficulty
3. Upload class image
4. Click **Save**

### Create Class Schedule
1. **Classes** > **Schedules** > **New Schedule**
2. Select class type
3. Choose recurrence (daily, weekly, specific days)
4. Set time and location
5. Assign trainer
6. Set capacity and booking window
7. Click **Create Schedule**

### Manage Individual Sessions
- **View**: Go to **Classes** > **Sessions**
- **Cancel**: Select session > **Cancel** > Notify members
- **Reschedule**: Select session > **Reschedule** > Set new time
- **Mark Attendance**: Select session > **Attendance** > Mark present/absent

### Handle Waitlists
1. Go to full class session
2. View waitlist
3. Manually promote members if spots open
4. Notify promoted members

---

## Attendance & Check-In

### Manual Check-In
1. **Attendance** > **Check-In**
2. Search member by name, phone, or member ID
3. Verify membership status
4. Click **Check-In**
5. Member checked in successfully

### Scan QR Code
1. Use webcam or QR scanner
2. Scan member's QR code from mobile app
3. Automatic check-in

### View Today's Attendance
- **Attendance** > **Today**
- See all check-ins with times
- Current facility occupancy
- Peak hours analysis

### Attendance Reports
- **Reports** > **Attendance**
- Filter by date range, club, member
- Export to Excel/PDF

---

## Invoicing & Payments

### Create Manual Invoice
1. **Invoices** > **New Invoice**
2. Select member
3. Add line items (description, price, tax)
4. Set due date
5. Click **Create**
6. Invoice automatically sent to member

### Record Payment
1. Open pending invoice
2. Click **Record Payment**
3. Select payment method (cash, card, bank transfer)
4. Enter amount
5. Add reference number
6. Click **Confirm**

### Payment Collection
- **Invoices** > Filter by "Overdue"
- Send payment reminders
- Call/email members with outstanding balance
- Record payments as received

### Refunds
1. Find paid invoice
2. Click **Refund**
3. Enter refund amount and reason
4. Select refund method
5. Confirm refund

---

## CRM & Lead Management

### Add Lead
1. **CRM** > **Leads** > **Add Lead**
2. Enter contact information
3. Select source (walk-in, referral, website, etc.)
4. Add notes
5. Assign to sales rep
6. Click **Save**

### Move Lead Through Pipeline
- **Pipeline View**: Drag and drop leads between stages
  - New → Contacted → Tour Scheduled → Trial → Negotiation → Won/Lost
- **List View**: Change status in lead details

### Log Activity
1. Open lead profile
2. Click **Add Activity**
3. Select type (call, email, meeting, tour)
4. Add notes
5. Schedule follow-up task
6. Save activity

### Convert Lead to Member
1. Open lead profile
2. Click **Convert to Member**
3. Select membership plan
4. Set start date
5. Create invoice
6. Lead becomes active member

---

## Marketing Campaigns

### Create Campaign
1. **Marketing** > **Campaigns** > **New**
2. Enter campaign name
3. Select type (one-time broadcast, drip, triggered)
4. Choose channel (email, SMS, WhatsApp, push)
5. Select target segment
6. Design message content
7. Schedule send time
8. Click **Create Campaign**

### Create Segment
1. **Marketing** > **Segments** > **New**
2. Set criteria:
   - Demographics (age, gender)
   - Membership status
   - Activity level
   - Engagement score
3. Preview member count
4. Save segment

### View Campaign Analytics
1. **Marketing** > **Campaigns**
2. Click on campaign
3. View metrics:
   - Sent, delivered, opened, clicked
   - Conversion rate
   - Revenue attributed

## Reports & Analytics

### Pre-Built Reports
- **Member Reports**: Active members, new signups, churn analysis
- **Financial Reports**: Revenue by period, outstanding payments
- **Attendance Reports**: Daily/weekly/monthly attendance, peak hours
- **Class Reports**: Popular classes, utilization, no-show rates
- **Sales Reports**: Lead conversion, sales by rep

### Generate Report
1. **Reports** > Select report type
2. Set date range
3. Apply filters (club, plan, status)
4. Click **Generate**
5. Export to PDF or Excel

### Schedule Reports
1. Open any report
2. Click **Schedule**
3. Set frequency (daily, weekly, monthly)
4. Add email recipients
5. Save schedule

## Troubleshooting

**Member can't log in**: Reset password from member profile
**Payment failed**: Check payment gateway status, retry payment
**Class full**: Add to waitlist or open additional session
**Equipment not syncing**: Check provider API connection status
**Report not generating**: Check date range and filters
