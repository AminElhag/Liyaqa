## QR Code Check-In

### Generate QR Code
1. Tap **Check-In** from home or bottom menu
2. Your unique QR code appears
3. Show to reception staff or scan at access device

### Features
- **Works Offline**: QR code generated locally, no internet needed
- **Auto-Refresh**: Updates every 60 seconds for security
- **Brightness Boost**: Screen automatically brightens for scanning
- **Member ID**: Shows your member number below QR code

### Troubleshooting
- **QR not scanning**: Increase screen brightness
- **Invalid QR**: Pull down to refresh
- **Access denied**: Check with reception (may be expired membership)

## Class Booking

### Browse Classes
1. Tap **Classes** from menu
2. View upcoming classes (next 7 days)
3. **Filter**: Tap filter icon
   - By date
   - By class type
   - By trainer
   - By time of day

### View Class Details
Tap any class card to see:
- Class name and description
- Instructor photo and name
- Date and time
- Duration (e.g., 60 minutes)
- Location (e.g., Studio A)
- Available spots (e.g., 5/20)
- Difficulty level
- Equipment needed

### Book a Class
1. Find desired class
2. Tap class card
3. Tap **Book Now** button
4. Confirm booking
5. See confirmation message
6. Booking added to "My Bookings"

### Cancel Booking
1. Go to **Classes** > **My Bookings** tab
2. Find booking to cancel
3. Swipe left on booking (iOS) or long press (Android)
4. Tap **Cancel**
5. Confirm cancellation
6. Receive cancellation confirmation

### Booking Reminders
- **24 hours before**: "Your Yoga class is tomorrow at 6:00 PM"
- **1 hour before**: "Your class starts in 1 hour. See you soon!"

## Personal Training

### Browse Trainers
1. Tap **More** > **Personal Training**
2. View all available trainers
3. See trainer details:
   - Photo
   - Name
   - Specializations (e.g., Weight Loss, Strength Training)
   - Years of experience
   - Languages spoken
   - Rate per session

### Request PT Session
1. Select a trainer
2. Tap **Book Session**
3. Choose preferred date
4. Select time slot
5. Add any notes or goals
6. Tap **Request Session**
7. Wait for trainer confirmation (usually within 24 hours)

### View PT Sessions
1. **More** > **PT Sessions**
2. **Upcoming**: See confirmed sessions
3. **Past**: View session history
4. Tap session for details

### Cancel PT Session
1. Go to **PT Sessions**
2. Tap session to cancel
3. Tap **Cancel Session**
4. Review cancellation policy
5. Confirm cancellation

## Subscriptions

### View Subscription
1. Tap **More** > **Subscriptions**
2. See active subscription details:
   - Plan name
   - Start and end dates

   - Guest passes available
   - Freeze days balance

2. View available freeze days

6. Add reason (optional)

3. Compare benefits

## Fitness Tracking

### Connect Wearable
1. **More** > **Fitness** > **Connect Device**
2. Select your wearable:
   - Fitbit
   - Garmin
   - Apple Health (iOS only)
   - Google Fit (Android only)
   - Strava
   - Others
3. Tap **Connect**
4. Authorize in wearable app
5. Return to Liyaqa app
6. Wait for initial sync

### View Activity
1. Tap **More** > **Fitness**
2. **Today** tab shows:
   - Steps
   - Calories burned
   - Active minutes
   - Distance
3. **Statistics** tab:
   - Average daily steps
   - Total calories (7/30/90 days)
   - Active hours
   - Sleep duration
4. **Workouts** tab:
   - Recent workouts from wearable
   - Duration, calories, heart rate

### Manual Sync
1. **Fitness** > Wearables
2. Find connected device
3. Tap **Sync Now**
4. Wait for sync complete

## Notifications

### Enable Push Notifications
1. When prompted on first launch, tap **Allow**
2. Or: **More** > **Settings** > **Notifications**
3. Enable **Push Notifications**

### Notification Types
- **Class Reminders**: 24h and 1h before class
- **Booking Confirmations**: Immediate after booking
- **Payment Due**: 3 days before and on due date
- **Subscription Expiry**: 30, 7, and 1 day before expiry
- **Freeze Approved**: When freeze request approved
- **Promotions**: Special offers and announcements

### Manage Notifications
- **Notification Center**: Tap bell icon to view all
- **Mark as Read**: Swipe or tap
- **Clear All**: Remove all read notifications

## Troubleshooting

**App won't load**:
- Check internet connection
- Force close and reopen app
- Update app to latest version

**Can't login**:
- Verify email and password
- Check caps lock
- Use "Forgot Password" to reset

**QR code not scanning**:
- Increase screen brightness
- Clean camera lens at scanner
- Pull down to refresh QR code

**Classes not loading**:
- Check internet connection
- Pull down to refresh
- Try closing and reopening app

**Notifications not working**:
- Check app notification settings
- Check device notification settings
- Ensure app is updated

**Wearable not syncing**:
- Check wearable app is installed
- Re-authorize connection
- Manually trigger sync
